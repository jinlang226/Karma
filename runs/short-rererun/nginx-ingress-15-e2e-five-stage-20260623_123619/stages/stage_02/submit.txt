HTTPS is working. The curl returns exit code 0 and prints "hello".

**Root cause:** The ingress `demo-ingress` referenced a secret named `expired-tls-secret` containing a certificate that expired in January 2020. The curl-test pod's CA trust (`/tmp/tls/ca.crt`) is `CN=Test CA` whose key lives at `/tmp/test-ca.key`, but the existing valid certs in `/tmp/` were signed by a *different* CA despite sharing the same CN.

**Fix applied:**
1. Generated a new 2048-bit key and signed a fresh certificate for `demo.example.com` using the correct `/tmp/test-ca.key` (valid 2026–2027, with the proper SAN)
2. Created/updated the `demo-tls-secret` TLS secret with the new cert+key
3. Patched `demo-ingress` to use `demo-tls-secret` instead of `expired-tls-secret`
