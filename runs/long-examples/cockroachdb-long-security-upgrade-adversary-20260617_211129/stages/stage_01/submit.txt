**Key decisions made:**
- `podManagementPolicy: Parallel` — required so all 3 pods start simultaneously (CockroachDB can't form quorum if nodes start sequentially waiting for each other)
- `publishNotReadyAddresses: true` on the headless service — required for DNS-based peer discovery before readiness probes pass
- `cockroach init` run once after nodes started — initializes the cluster bootstrapping
