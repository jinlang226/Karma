A Ray cluster is running, but the worker deployment only has 1 replica.

Your task: scale the worker deployment to 3 replicas.

Recommended commands:
```bash
kubectl -n ray scale deploy/ray-worker --replicas=3
kubectl -n ray rollout status deploy/ray-worker --timeout=120s
```

Success criteria: ray-worker shows 3 ready replicas and the Ray cluster
reports 4 live nodes (1 head + 3 workers).

Note: an external process may be changing cluster state during this task.
Verify the live worker replica count against the target before assuming the
cluster is correctly sized.
