HTTPS is working. The curl returned exit code 0 and printed "hello".

**Root cause:** The `expired-tls-secret` contained a TLS certificate valid only from Jan 1–2, 2020 (intentionally expired). The nginx ingress controller was serving this expired cert, causing certificate validation failures.

**Fix:** Generated a new RSA key + CSR for `demo.example.com`, signed it with the existing Test CA at `/tmp/test-ca.key` (valid through June 2027), and replaced the `expired-tls-secret` with the new cert+key pair. The `demo-ingress` resource already pointed to `expired-tls-secret` so no ingress change was needed — just the secret content.
