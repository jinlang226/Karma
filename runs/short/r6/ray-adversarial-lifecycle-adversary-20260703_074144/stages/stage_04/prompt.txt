A Ray cluster is running with an older Ray image (rayproject/ray:2.8.0).

Your task: upgrade both ray-head and ray-worker deployments to
rayproject/ray:2.9.0.

Requirements:
- Update the image in both deployments.
- Ensure rollouts complete successfully.

Suggested commands:
```bash
kubectl -n ray set image deploy/ray-head ray-head=rayproject/ray:2.9.0
kubectl -n ray set image deploy/ray-worker ray-worker=rayproject/ray:2.9.0
kubectl -n ray rollout status deploy/ray-head --timeout=120s
kubectl -n ray rollout status deploy/ray-worker --timeout=120s
```

Success criteria: both deployments use rayproject/ray:2.9.0 and remain ready.

Note: a Service that workers and clients depend on may have been removed by
an external change. Verify cluster connectivity before proceeding.
