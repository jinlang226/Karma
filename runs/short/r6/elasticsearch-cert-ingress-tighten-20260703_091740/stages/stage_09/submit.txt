4. **`es-config` ConfigMap updated** — Added three static S3 client settings (safe to put in YAML — these are not credentials):
   - `s3.client.minio.endpoint: minio.elasticsearch.svc.cluster.local:9000`
   - `s3.client.minio.protocol: http`
   - `s3.client.minio.path_style_access: true`

5. **StatefulSet patched** — Added a `keystore-setup` init container that runs on every pod start to:

   - Create a fresh keystore in the emptyDir and populate it with the S3 credentials from the `es-secure-settings` secret
   - The emptyDir is mounted at `/usr/share/elasticsearch/config/` in the main container (with subPath mounts for elasticsearch.yml, certs, and file-realm layering on top)

6. **Snapshot repository registered** — `minio-repo` of type `s3` targeting the `es-backups` bucket using the `minio` client — all 3 nodes verified connectivity.
