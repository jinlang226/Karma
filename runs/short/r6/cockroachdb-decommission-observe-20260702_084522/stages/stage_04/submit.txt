**DB Console (HTTPS)**

- Generated a self-signed TLS certificate for `crdb-ui.example.com` and stored it as secret `crdb-ui-tls` in the `cockroachdb` namespace.
- Created `Ingress/crdb-ui` (class `nginx`) routing `https://crdb-ui.example.com/` → `crdb-cluster-public:8080`. nginx terminates TLS and proxies plain HTTP to the backend (CRDB runs `--insecure`). Verified: HTTP 200.

**SQL TCP (port 26257)**

- Created ConfigMap `ingress-nginx/tcp-services` with entry `"26257": "cockroachdb/crdb-cluster-public:26257"`.
- Patched the `ingress-nginx-controller` Deployment to add `--tcp-services-configmap=ingress-nginx/tcp-services`.
- Patched the `ingress-nginx-controller` Service to expose port 26257 (NodePort 32254).
- Verified: `cockroach sql --host=<controller-ip> --port=26257 --insecure` returns results.
