HTTPS is restored. The curl returns exit code 0 and prints "hello".

**Root cause:** The Ingress `demo-ingress` referenced `expired-tls-secret`, which contained a certificate for `demo.example.com` that expired on January 2, 2020 — valid for only one day.

**Fix:** Used the Test CA private key at `/tmp/test-ca.key` to issue a new leaf certificate for `demo.example.com` (valid through 2028) with the correct SAN, then updated `expired-tls-secret` in-place. The nginx ingress controller hot-reloaded the new cert automatically.
