HTTPS is working. The response is `hello` with exit code 0 and `SSL certificate verify ok`.

**Root cause:** The ingress `demo-ingress` referenced a secret named `expired-tls-secret` containing a certificate that expired in January 2020 (valid only Jan 1–2, 2020).

**Fix applied:**
1. Generated a new RSA 2048 key + certificate for `demo.example.com` signed by the test CA (`/tmp/test-ca.crt` + `/tmp/test-ca.key`), valid until October 2028.
2. Created a new secret `demo-tls-valid` in the `demo` namespace.
3. Patched the ingress to use `demo-tls-valid` instead of `expired-tls-secret`.
