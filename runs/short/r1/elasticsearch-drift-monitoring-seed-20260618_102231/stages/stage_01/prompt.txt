A new Elasticsearch cluster must be deployed in the elasticsearch namespace.

OBSERVED STATE:
- Namespace elasticsearch exists.
- No Elasticsearch Services, StatefulSets, PDBs, or ConfigMaps exist.
- TLS secret es-http-tls exists with tls.crt, tls.key, and ca.crt.
- Password secret elastic-password exists with key "password".

YOUR TASK:
Deploy a 3-node cluster using only native Kubernetes resources.

CONSTRAINTS:
- Do not install CRDs or use operators.
- Use image docker.elastic.co/elasticsearch/elasticsearch:8.11.1.

GOAL:
- All nodes are Ready.
- Elasticsearch is reachable over HTTPS (port 9200) inside the cluster through a
  Service named es-http.
- Reads and writes succeed.
