Deployed a 3-node Elasticsearch 8.11.1 cluster in the elasticsearch namespace using native Kubernetes resources.

1. Generated TLS certificates (CA + server cert with proper SANs) using OpenSSL locally.
2. Created secrets: es-http-tls (tls.crt, tls.key, ca.crt) and elastic-password (password=elasticpass).
