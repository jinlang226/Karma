Enable TLS for the running CockroachDB cluster in the cockroachdb namespace.

The cluster is currently running in insecure mode. An openssl-toolbox pod
is available for certificate generation.

GOAL:
1. Insecure access is rejected:
   ```bash
   kubectl -n cockroachdb exec crdb-cluster-0 -- \
     ./cockroach sql --insecure -e "SELECT 1;"
   ```
2. Secure access succeeds:
   ```bash
   kubectl -n cockroachdb exec crdb-cluster-0 -- \
     ./cockroach sql --certs-dir=/cockroach/cockroach-certs -e "SELECT 1;"
   ```
